Lab 1: Ansible - Install Nginx Step 1: Launch a Linux Server

Use:

Amazon Linux 2/2023 or RHEL or CentOS

Check the OS:

cat /etc/os-release

Example output:

NAME="Amazon Linux" VERSION="2023" Step 2: Install Ansible

For Amazon Linux:

sudo dnf install ansible -y

For RHEL/CentOS 7:

sudo yum install epel-release -y sudo yum install ansible -y

Check the version:

ansible --version

Expected output:

ansible [core 2.x.x] Step 3: Create a Project Directory mkdir ansible-project cd ansible-project

Check:

pwd Step 4: Create an Inventory File vi inventory

Add:

[web] localhost ansible_connection=local Why?

Normally, Ansible connects to remote servers using SSH.

Since you're practicing on the same machine, ansible_connection=local tells Ansible to run locally.

Step 5: Create the Playbook vi install-nginx.yml

Paste:

name: Install Nginx on Pathnex server hosts: web become: yes

tasks:

name: Install nginx yum: name: nginx state: present
Save and exit.

Step 6: Check the YAML Syntax ansible-playbook --syntax-check install-nginx.yml

Expected:

playbook: install-nginx.yml

No errors mean the syntax is correct.

Step 7: Run the Playbook ansible-playbook -i inventory install-nginx.yml

Ansible will:

Read the inventory. Find the web host. Connect locally. Become root. Install Nginx. Step 8: Verify

Check if Nginx is installed:

nginx -v

Or:

rpm -qa | grep nginx Step 9: Start Nginx sudo systemctl start nginx

Enable it at boot:

sudo systemctl enable nginx

sudo systemctl status nginx What You Learned Install Ansible Create an inventory Write a playbook Run a playbook Install software automaticallyLab 1: Ansible - Install Nginx Step 1: Launch a Linux Server
